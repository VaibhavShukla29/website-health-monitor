# ── httpbin.org — free HTTP testing service ────────────────────────────────
# These endpoints let you simulate every scenario without needing a broken site.

# Always returns HTTP 200 — simulates a healthy website
https://httpbin.org/status/200

# Always returns HTTP 404 — SERVER IS UP, just no page at this path
# Your monitor reports this as: UP | 404
# Teaches the UP/DOWN distinction (UP = infrastructure alive)
https://httpbin.org/status/404

# Always returns HTTP 500 — application error, but server is responding
# Your monitor reports this as: UP | 500
https://httpbin.org/status/500

# Delays response by 3 seconds — tests slow website detection
# You should see "Slow" in the performance column
https://httpbin.org/delay/3
