# requirements.txt — Python Dependencies

# Website Health Monitoring System

# Install all dependencies:
#   pip install -r requirements.txt
#
# Why pin versions (requests==2.31.0)?
#   Unpinned 'pip install requests' installs the latest version.
#   New versions can introduce breaking changes.
#   Pinned versions ensure reproducible builds across every developer
#   machine, CI/CD pipeline, and Docker container.
#   This is a core DevOps practice.
#
# Regenerate this file from your current environment:
#   pip freeze > requirements.txt
